ubuntu下安装与配置phpMyAdmin

mysql

ubuntu下安装和配置phpMyAdmin非常的简单。
安装: 

sudo apt-get install phpmyadmin

在安装过程中会要求输入几次密码，按照提示输入就好。

安装完成后将/usr/share/ 下的phpmyadmin文件夹复制到/var/www/ 下。 

cp -r /usr/share/phpmyadmin /var/www/

然后在浏览器中输入http://localhost/phpmyadmin即可访问。

如果出现The mysqli extension is missing.的错误，运行命令 

sudo /etc/init.d/apache2 restart

再次打开即可。
